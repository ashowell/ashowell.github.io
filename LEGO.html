Project Overview

The purpose of this project was to design a LEGO Architecture assembly. My choice of Copenhill ("Amager Bakke") by Bjarke Ingels was inspired by his ability to create sustainably.

After creating the cad model for a single 2x4 white brick, a design table was used to auto generate all other bricks needed given few inputted values.
